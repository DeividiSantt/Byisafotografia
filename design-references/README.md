# Direção Visual

## Behance Van Gogh Website Design UI

Referência usada apenas para direção de composição: blocos assimétricos, imagens com formas orgânicas, hierarquia com títulos grandes, contraste de azul profundo com amarelo e sobreposições controladas. O conteúdo, textos, imagens e identidade desta landing foram criados para Isadora.

## Fotos anexadas

As imagens foram organizadas em `public/images` para uso real no hero, seção sobre, portfólio, serviços e homenagem. Os caminhos editáveis do portfólio ficam em `src/data/portfolio.ts`.

## Paleta

Base em azul noturno, azul intenso, amarelo girassol, oliva, laranja queimado e branco quente. O amarelo entra como acento para evitar aparência infantil ou comemorativa demais.
